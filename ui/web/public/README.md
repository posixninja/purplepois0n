# Brand assets

Replace these files with your official purplepois0n artwork:

| File | Use |
|------|-----|
| `logo.svg` | Sidebar, page heroes, empty states |
| `favicon.svg` | Browser tab icon |

Recommended: SVG with transparent background, square or ~1:1 mark. The UI scales the logo to 32–48px; keep detail readable at small sizes.

After swapping assets, run `npm run build` or refresh the dev server.
